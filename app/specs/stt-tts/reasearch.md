Android provides built-in modules for Speech-to-Text (STT) and Text-to-Speech (TTS) primarily through the  package and Google Services. Key tools include the SpeechRecognizer API for voice transcription and TextToSpeech class for voice synthesis, allowing on-device, low-latency audio processing without third-party libraries. Built-in Android STT (Speech-to-Text) 

• SpeechRecognizer API: The native Android API used to convert spoken audio into text. 
• SpeechRecognizerIntent: Enables intents to launch the standard Google voice recognition dialog. 
• Gboard Voice Typing: The default keyboard engine that provides offline and online speech recognition. 
• Key Capabilities: Requires  permission. [1, 7, 8, 9, 10]  

Built-in Android TTS (Text-to-Speech) 

• TextToSpeech (TTS) API: A standard Android class that allows apps to synthesize speech from text. 
• Google TTS Engine: The default engine powering voice output across Android, accessible via . 
• Key Capabilities: Supports multiple languages, voice pitch adjustment, and speech rate control. [2, 3, 11, 12, 13]
